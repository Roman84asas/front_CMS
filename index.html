Используется для доставки виртуальных рабочих мест с применением различных протоколов удаленного доступа — SPICE, RDP, HTML5, x2GO, VNC и других.

Диспетчер подключений совместим с различными гипервизорами, операционными системами и предоставляет управляемый доступ к виртуальным рабочим местам на основе заданных параметров конфигурации.

Включен в единый реестр российских программ для электронных вычислительных машин и баз данных. Рег. номер ПО: 5317
Диспетчер подключений виртуальных рабочих мест
ХАРАКТЕРИСТИКИ
Описание основных возможностей диспетчера подключений при реализации полного жизненного цикла виртуальных рабочих мест пользователей
  • Управление виртуальными машинами на гипервизоре: создание, размещение, запуск, выключение, сохранение состояния;
  • Работу c любым гипервизором: Linux KVM, VMware, Citrix, Hyper-V;
  • Поддержку гостевых операционных систем: MS Windows XP/7/8/10/Server; GNU/Linux Ubuntu, Debian, CentOS, Alt, Astra, Rosa, Gos и других;
  • Управление доставкой виртуального рабочего места;
  • Формирование индивидуальных и групповых фондов виртуальных рабочих мест;
  • Доступ к рабочим местам из веб-обозревателя или специализированных клиентов;
  • Поддержку спектра протоколов доставки виртуальных рабочих мест: SPICE, RDP, HTML5, x2GO, VNC и других;
  • Назначение пользователю доступных для него рабочих пространств из фонда виртуализированных рабочих мест;
  • Полный спектр методов аутентификации: локальная база данных, MS Active Directory, LDAP/OpenLDAP, включая Single Sign-On;
  • Различные способы идентификации: по имени пользователя, по именам устройств, по IP адресам, по MAC адресам;
  • Управление разграничением прав доступа к периферийным устройствам: звук, USB, принтеры, ключи, смарткарты;
  • Интеграцию с широким спектром аппаратных архитектур: ARM, MIPS, x86 и других;
  • Мониторинг состояния фондов рабочих мест с возможностью удаленного контроля рабочего пространства;
  • Предоставление накопленных и оперативных статистических данных о работе инфраструктуры виртуальных рабочих столов.
